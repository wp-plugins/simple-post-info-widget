== Description ==

By simply entering your posts ID number you can directly display your specific post information on your WordPress sidebar in widget form.

Post information currently includes:

*Widget title (overrides post title)
*Featured image
*Title
*Excerpt
*Link directly to post

It is a very stripped down plugin enabling you to style it up yourself in accordance with your sites branding and theme.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2 Go to plugins and activate plugin
3 Go to appearance > widgets and find your new widget here.
